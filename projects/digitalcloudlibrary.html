Digital Cloud Library

As a software developer at VCL in Bangalore, I worked majorly on Qt based GUI development for a digital cloud library application. This software is configured in various discovery stations at libraries which encourages patrons to browse and check out the digital content. A staff library member lends the e-reader compatible with the cloud library supported e-reader device to the patron. This software application is synchronized with e-reader device and it transfers checked out book from a patron's online account to the device. In a way, this e-book lending solution makes books accessible to people who don't have their own devices.

Technologies Used

C++ / Qt Framework, QML, JSON web services, Adobe RMSDK
